# 本田圭佑とじゅいんいちダビッドソン判別アプリ

## URL
https://hondarecognition-app.herokuapp.com/

## 概要
画像を読み込んで本田圭佑とじゅんいちダビッドソンを判別し、同時にその判別した判定結果の確率を算出する
